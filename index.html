Why care?

Everyone in every walk of life should be able to use our digital products.

Because a label is provided in the button, we should treat this icon as decorative. We add an "aria-hidden" attribute to hide the SVG from screen readers. There's one more thing we need to do to make this bulletproof.
We add a "focusable='false'" attribute to ensure IE won't give it focus.

Four Steps

  1. Use SVG
  2. Set the focusable attribute to "false" on the SVG
  3. Set the aria-hidden attribute to "true" on the SVG
  4. If the button doesn't have a label, give it one with the aria-label attribute
