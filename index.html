Vue Ramp-up

What is Vue.js?

Vue (pronounced like "view") is a JS framework for building user interfaces.

How does it compare to React?

Similarities

  • utilize a virtual DOM
  • provide reactive and composable components
  • use one-way data flow
  • rely on other libraries to do things like routing

Differences

  • templates instead of JSX
  • styles defined with and scoped to components
  • generally more "magical"
